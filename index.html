Lowersrc

Fakey images for prototypin'

The idea is to use canvas to create fake images as placeholders in designs. So when you are mocking stuff up you can quickly add fake images - like loren ipsum for images.

To use it, you include a regular img tag, but with a width, height, and make faker image. lol-d-lol.

(proposed) Features:

  • Options to customise the images:
    • rounded corners
    • styles
    • colours
    • procedural patterns
  • Colour schemes
  • Set "defaults" for all images

Create a weird montage from parts of images to make "loren ipsum" of people for photo-like images

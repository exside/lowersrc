lowersrc

Loren Ipsum for images

Easily create fake images as placeholders for live protyping designs. Simply add regular-old HTML image tags - lowersrc will dynamically create the contents of the image (based on your customisations) so you can get on with the layout.

Add the lowersrc script to your project, and add an image with a "lowersrc" class. The default image is a rectangle with a thin cross through it - but you can add text, change colours, change patterns - whatever works for your wireframe. Because they are rendered as regular HTML images, they can also be styled and targeted like regular images. They can be any size you need.

Using lowersrc

Add a regular ol' image tag, and give it the "lowersrc" class. You can customise the output by setting attributes:

<img class="lowersrc" />
<img class="lowersrc" width="100" height="150" />
<img class="lowersrc" data-text="My Image" />

Features

  • Options to customise the images:
    • Background: data-bg="none|solid" data-bg-col="#f0f|{random}"
    • Border: data-border="none|solid" data-border-col="#f0f|{random}"
    • Foreground: data-fg="none|cross|circle" data-fg-col="#f0f|{random}"
    • Text: data-text="your text" data-text-col="#f0f|{random}" data-text-bg-col="#f0f|{random}"
    • Colours: can be any valid HTML colour, or the string "{random}" for a random colour
  • Set some "defaults" for all images based on the setting of one image (add the attribute data-default="true" to one of the images)
  • Allow % dimensions

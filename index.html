lowersrc

Loren Ipsum for images

Easily create fake images as placeholders for live protyping designs. Simply add regular-old HTML image tags - lowersrc will dynamically create the contents of the image (based on your customisations) so you can get on with the layout.

To use it, you include a regular img tag, but with a width, height, and make faker image. lol-d-lol.

Using lowersrc

Add a regular ol' image tag, and give it the "lowersrc" class. You can customise the output by setting attributes:

<img class="lowersrc" />
<img class="lowersrc" width="100" height="150" />
<img class="lowersrc" data-text="My Image" />

Features

  • Options to customise the images:
    • colours
  • Colour schemes
  • Set "defaults" for all images
